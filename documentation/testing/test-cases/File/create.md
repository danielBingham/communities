## [Create File](documentation/testing/test-cases/File/create.md)

Cases covering making posts in all their forms and with all their attachments.

### Pre-requisites

- [ ] User1 has been created.
- [ ] User2 has been created.

### Cases

#### Post

- [ ] As a user, I can upload an image to a post.
- [ ] As a user, I can upload several images to a post.
- [ ] As a user, I can upload a video to a post.
- [ ] As a user, I can upload several videos to a post.
- [ ] As a user, I can upload both videos and images to a post.
- [ ] As a user, I can upload up to 30 items to a post.
- [ ] As a user, I can reorder items I uploaded and have the order respected.

- [ ] As a user, I get a clear error message when an upload error occurs.
- [ ] As a user, I get a clear error message when I try to upload an image larger than 30 mb.
- [ ] As a user, I get a clear error message when I try to upload a video larger than 700 mb.
- [ ] As a user, I get a clear error message when I try to upload an empty file.
- [ ] As a user, I get a clear error message when a network error occurs.
- [ ] As a user, I get a clear error message when a processing error occurs.

#### Group Profile

- [ ] As a user, I can crop the image I uploaded.

- [ ] As a user, I get a clear error message when I try to upload a video larger than 700 mb.

- [ ] As a user, I get a clear error message when a network error occurs.
- [ ] As a user, I get a clear error message when a processing error occurs.
